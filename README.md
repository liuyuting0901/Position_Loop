# Position_Loop
双电机动捕位置环

## 文件结构

| 文件 | 功能说明 |
|:---|:---|
| `Main.py` | 主程序，状态机逻辑、数据记录与可视化绘图 |
| `Controller.py` | 闭环位置/速度串级 PID 控制器，含积分限幅与油门滤波 |
| `MoCapHandler.py` | 动捕数据读取线程（基于 LuMo SDK），坐标系转换与姿态低通 |
| `KalmanFilter.py` | 3D 常速度线性卡尔曼滤波器，含异常跳变剔除 |
| `MavLinkBridge.py` | MAVLink 桥接，发送姿态/油门指令，监控心跳与飞控模式 |

### 1. 油门搜索起飞
扑翼机每次起飞受电池电压、扑频、翼面效率、地面摩擦等影响，固定起飞油门极易失败。本系统采用**渐进式油门搜索**：
- 起飞阶段不让目标高度独自爬升（避免位置环饱和）；
- 从安全油门起步，按速率线性递增，直到检测到离地；
- 离地后自适应估算悬停油门初值，防止冲顶。

### 2. 抗扑翼抖动设计
- **卡尔曼滤波**：滤除动捕高频噪声，输出平滑的位置与速度估计；
- **油门低通滤波 + 变化率限制**：抑制由拍翼引起的油门高频抖动与突变；
- **保守的 PID 参数**：速度环微分项通常设为0，避免差分放大噪声。

### 3. 完善的安全保护
- 动捕数据丢失/跳变保护；
- MAVLink 心跳超时保护；
- 高度超限（冲顶）保护；
- 起飞油门搜索超时保护；
- Ctrl+C 中断自动进入降落/低油门状态。

## 飞行状态机
| 状态 | 说明 |
|:---|:---|
| **IDLE** | 程序启动，等待动捕锁定与用户确认 |
| **THRUST_SEARCH** | 油门搜索：水平姿态 + 缓慢递增油门，直到检测离地 |
| **ASCENT** | 闭环爬升：离地确认后，从当前高度平滑爬升至目标悬停高度 |
| **HOVER** | 定高悬停：到达目标高度后闭环悬停，允许 Z 轴积分修正悬停油门 |
| **LANDING** | 正常降落：斜坡下降目标高度，关闭积分防触地反弹 |
| **EMERGENCY** | 紧急保护：动捕丢失/高度超限/链路超时，直接发送水平姿态+低油门 |
